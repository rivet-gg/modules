## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add tokens
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"tokens": {}
}

```
</Tab>

</Tabs>
